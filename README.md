# 📚 Lógica de Programação

Este repositório reúne exercícios e atividades desenvolvidos durante meus estudos de **Lógica de Programação**, principalmente ao longo da **Residência em TIC – Software do Serratec**.

A proposta deste repositório é registrar minha evolução na programação, desde os primeiros contatos com algoritmos até exercícios mais estruturados envolvendo vetores, matrizes e funções. Aqui estão armazenadas as atividades realizadas durante as aulas e práticas propostas ao longo da residência.

---

# 🎯 Objetivo do Repositório

Este repositório foi criado com alguns objetivos claros:

* Registrar minha evolução durante a **Residência Serratec**
* Organizar os exercícios de lógica desenvolvidos ao longo das aulas
* Servir como material de revisão para estudos futuros
* Demonstrar minha base de raciocínio lógico em programação

Além disso, ele também funciona como parte do meu **portfólio de aprendizado** na área de tecnologia.

---

# 🧠 Conteúdos Trabalhados

Os exercícios presentes neste repositório exploram conceitos fundamentais da programação, como:

### Fundamentos

* Variáveis
* Tipos de dados
* Operadores aritméticos e lógicos
* Entrada e saída de dados

### Estruturas de controle

* Condicionais (`se`, `senão`)
* Estruturas de repetição (`para`, `enquanto`)

### Estruturas de dados

* Vetores
* Matrizes
* Manipulação de dados em estruturas

### Organização do código

* Funções
* Separação de lógica em partes reutilizáveis
* Introdução à recursividade

Esses conceitos são essenciais para quem está começando na programação e fazem parte da base ensinada durante a residência.

# 📈 Processo de Aprendizado

Durante a residência do **Serratec**, os conteúdos são construídos de forma progressiva. Primeiro são apresentados os conceitos mais básicos da programação e, aos poucos, os exercícios passam a exigir mais organização e raciocínio lógico.

Esse repositório acompanha justamente essa evolução: desde exercícios simples de entrada e saída de dados até problemas que exigem estruturas de repetição, vetores, matrizes e funções.
